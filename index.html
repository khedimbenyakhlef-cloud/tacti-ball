Documentation API DeepSeek

Intégration DeepSeek

Notre plateforme intègre l'API DeepSeek pour l'analyse tactique avancée. Voici comment l'utiliser :

1
Obtenir une clé API

Visitez le site DeepSeek pour obtenir votre clé API gratuite.

2
Configurer l'intégration

Ajoutez votre clé API dans les paramètres de votre compte.

3
Utiliser l'analyse IA

Téléchargez des images/vidéos pour une analyse tactique complète.

Exemple de Requête DeepSeek

// Requête à l'API DeepSeek
const response = await fetch('/api/deepseek/analyze', {
    method: 'POST',
    headers: {
        'Authorization': 'Bearer VOTRE_CLE_API',
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        image: 'base64_encoded_image',
        analysis_type: 'tactical',
        options: {
            detect_players: true,
            detect_formation: true,
            analyze_movement: true
        }
    })
});

// Réponse de l'API
const result = await response.json();
console.log(result);
